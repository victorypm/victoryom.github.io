创建Git仓库

登录https://github.com，点击右上角头像图标左面的加号，然后选择new repository， 出现如下页面：

填入仓库名称xxxx.github.io，注意此处的xxxx必须跟Github的账号名称一致，比如我的Github账号为victorypm，则仓库名称应该为victorypm.github.io，然后填入Description(可选)，最后点击Create repository。现在仓库就成功创建了
